# Icons TODO

The following icon files should be created for the add-on:

## Required Icons

### icon.png
- Size: 256x256 pixels
- Format: PNG
- Description: Small icon displayed in the add-on list
- Design suggestion: Game controller or keyboard icon with a remote/wireless symbol

### logo.png
- Size: 512x512 pixels (or larger, up to 1024x1024)
- Format: PNG
- Description: Larger logo displayed on the add-on details page
- Design suggestion: Same as icon.png but higher resolution

## Design Ideas

- Central element: Keyboard/remote/gamepad icon
- Accent: Wireless/Bluetooth waves or signal icon
- Color scheme: Blue (#03a9f4) to match the branding
- Style: Flat, modern, clean

## Creation Options

1. Use a design tool (Figma, Inkscape, GIMP)
2. Use an AI image generator
3. Commission from a designer
4. Use open-source icons from:
   - Material Design Icons (https://materialdesignicons.com/)
   - Font Awesome
   - The Noun Project
